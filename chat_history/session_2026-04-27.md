### Generátor webu (lekce 34)
- Čte `.py` soubory → generuje HTML s syntax highlighting
- `web/index.html` – live search, progress tracker (localStorage), dark/light toggle, copy kód
- Spustit: `python3 34_generator_webu.py`

### Vyřešené problémy
1. `class="cm">` viditelný jako text → `sub_mimo_spany()` v highlighteru
2. Soubory s diakritikou (05_podmínky.html) → `ascii_stem()` funkce
3. GitHub Pages 404 → `gh api PUT build_type=workflow`
4. Úlohy šedě v code bloku → `odstran_ulohy_z_kodu()`

### Agent konfigurace pro paralelní běh
- `CLAUDE.md` – kontext pro každý Claude Code worker
- `.claude/agents/` – lesson-writer, web-fixer, repo-maintainer
- `.claude/managed-agents/` – YAML + `parallel-session.py` pro N agentů najednou
